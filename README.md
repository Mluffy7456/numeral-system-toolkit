# 🔢 Numeral System Toolkit

A console application written in Python for working with numeral systems, arithmetic calculations, and bitwise operations.

---

## ✨ Features

### 🔄 Number Converter

- Convert numbers between numeral systems
- Supports bases **2–36**
- Supports positive and negative numbers
- Input validation

### 🧮 Calculator

Supported operations:

- Addition
- Subtraction
- Multiplication
- Integer division
- Modulo

Works in numeral systems from **2 to 36**.

### ⚙️ Bitwise Operations

Supported operations:

- AND
- OR
- XOR
- NAND
- NOR
- NOT

Configurable bit width:

- 8-bit
- 16-bit
- 32-bit
- 64-bit

### 📜 History

- JSON storage
- View history
- Clear history

### ✅ Validation

- Base validation
- Number validation
- Invalid character detection
- User-friendly error messages

# 🚀 Installation

Clone repository

```bash
git clone https://github.com/Mluffy7456/numeral-system-toolkit.git
```

Open project

```bash
cd numeral-system-toolkit
```

Run

```bash
python main.py
```
