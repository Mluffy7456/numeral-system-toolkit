# 🔢 Numeral System Toolkit

A feature-rich Python CLI toolkit for numeral system conversion, arithmetic calculations, bitwise operations, and digital logic.

---

## ✨ Features

### 🔄 Number Converter

- Convert numbers between numeral systems
- Supports bases **2–36**
- Supports positive and negative numbers
- Input validation
- Automatic result representations

---

### 🧮 Calculator

Supported operations:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Integer Division (`/`)
- Modulo (`%`)

Works with numeral systems from **2 to 36**.

---

### ⚙️ Bitwise Operations

Supported operations:

- AND
- OR
- XOR
- NAND
- NOR
- NOT
- Shift Left (`<<`)
- Shift Right (`>>`)
- Rotate Left (`ROTL`)
- Rotate Right (`ROTR`)

Configurable bit width:

- 8-bit
- 16-bit
- 32-bit
- 64-bit

---

### 📊 Number Representations

After every conversion or calculation, the result is automatically displayed in:

- Binary
- Octal
- Decimal
- Hexadecimal

---

### 📜 History

- JSON-based storage
- Rich table view
- View history
- Clear history

---

### 🎨 Rich CLI Interface

- Colored output
- Rich tables
- Panels
- Better error messages
- Improved readability

---

### ✅ Validation

- Base validation
- Number validation
- Invalid character detection
- User-friendly error messages

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Mluffy7456/numeral-system-toolkit.git
```

Open the project

```bash
cd numeral-system-toolkit
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
