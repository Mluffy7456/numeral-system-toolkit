# Numeral System Toolkit

A console application written in Python for working with numeral systems.

## Features

- Number converter
- Numeral system calculator
- Input validation
- Calculation history
- Bitwise operations (planned)

## Technologies

- Python 3

## Project Structure

```
NumeralSystemToolkit/

main.py
converter.py
calculator.py
validator.py
history.py
utils.py
config.py
```

## Installation

```bash
git clone https://github.com/your_username/numeral-system-toolkit.git
```

```bash
cd numeral-system-toolkit
```

```bash
python main.py
```
