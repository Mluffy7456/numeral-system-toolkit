# 🔢 Numeral System Toolkit

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Version](https://img.shields.io/badge/version-1.2.0-success)
![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-stable-brightgreen)

A modern Python CLI toolkit for numeral system conversion, arithmetic calculations, bitwise operations, and digital logic.

---

## ✨ Features

### 🔄 Number Converter

- Convert numbers between numeral systems
- Supports bases **2–36**
- Supports positive and negative numbers
- Automatic validation
- Automatic number representations

---

### 🧮 Calculator

Supports arithmetic operations in any numeral system:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Integer Division (`/`)
- Modulo (`%`)

Supports numeral systems from **2 to 36**.

---

### ⚙️ Bitwise Operations

Supported operations:

- AND
- OR
- XOR
- NAND
- NOR
- NOT
- Shift Left (`<<`)
- Shift Right (`>>`)
- Rotate Left (`ROTL`)
- Rotate Right (`ROTR`)

Configurable bit width:

- 8-bit
- 16-bit
- 32-bit
- 64-bit

---

### 📊 Number Representations

Results are automatically displayed in:

- Binary
- Octal
- Decimal
- Hexadecimal

---

### 📜 History

- JSON-based history
- Rich table view
- View history
- Clear history

---

### 🌍 Localization

Supported languages:

- 🇺🇸 English
- 🇷🇺 Русский

Language preference is automatically saved.

---

### 🎨 Rich CLI

Powered by **Rich** for:

- Colored output
- Tables
- Panels
- Better readability
- Improved user experience

---

### ✅ Validation

- Base validation
- Number validation
- Invalid symbol detection
- User-friendly error messages

---

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Mluffy7456/numeral-system-toolkit.git
```

Enter the project directory

```bash
cd numeral-system-toolkit
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
